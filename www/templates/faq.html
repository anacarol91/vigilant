Eu não possuo um veículo. Posso utilizar o aplicativo?
Mas é claro! O Vigilant foi feito para todas as pessoas com espírito colaborativo! Basta indicar na hora do cadastro que não você não possui um veículo.

Quais são as denúncias que eu possuo acesso?
As denúncias são filtradas de acordo com sua localização. Então é muito importante que você habilite o GPS para uma melhor experiência. Quais os dos meus dados são compartilhados com outros usários?
Nenhum dos seus dados é disponibilizado para a rede de usuários. Todas as denúncias são feitas de forma anônima, visando proteger a privacidade dos usuários.

Eu acredito que criaram uma denúncia para o meu carro de maneira errônea. Como posso proceder?
Vá até a denúncia, e reporte como spam. Digite uma justificativa clara, e então um de nossos administradores irá avaliar a vericidade da denúncia. Caso ela seja considerada falsa, será apagada.

Não quero continuar usando o Vigilant. Os meus dados continuarão na base de dados do aplicativo?
Ao encerrar sua conta, todos os seus dados são apagados do nosso servidor. Essa é uma operação irreversível.
